# Start Git
Command for start git

Clone Project

 > git clone https://url.git
 
 or Set a new remote
 
 > git remote add origin https://url.git
 
 > cd project_folder
 
 > echo "#Project Name" >> README.md
 
 > echo ignore_folder >> .gitignore
 
 > git add .
 
 > git commit -m "Initial Project"
 
 > git push -u origin master
